
  # AI-Powered Resume Screening & Matching System

**Intern:** Supriya Tandukar
**Date:** June 2026

---

## Business Problem
Companies receive hundreds of resumes per job posting. Manually screening
them is slow and inconsistent. This tool automates matching resumes to job
descriptions and explains why a candidate is a good fit or not.

---

## What I'm Building
A Streamlit web app that:
- Takes a job description and resume as input
- Predicts a match score for the candidate
- Uses an LLM to generate a 3-bullet candidate summary
  and a missing skills report
- Includes a chatbot (Week 6) to query the resume database

---

## Dataset
- **Resumes:** Resume Dataset on Kaggle (2,400+ resumes, 25 job categories)
https://www.kaggle.com/datasets/snehaanbhawal/resume-dataset
- **Job Descriptions:** 15 synthetic JDs generated using ChatGPT

---

## Success Metrics
